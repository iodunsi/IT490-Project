This branch is dedicated to deployment. See deployment folder.

# Deployment Scripts
- `setup_env.sh`: Configures new VMs for prod/QA.
- `package.sh`: Creates versioned package.
- `deploy.sh`: Deploys to QA/prod.
- `verify.sh`: Verifies deployment.

# Scripts to add later when environments are cloned:
- `mark_bad.sh`: Marks package as bad.
- `rollback.sh`: Reverts to previous version.
